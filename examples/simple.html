EasyModal - 简单示例

这是一个最简单的使用示例，展示如何在你的项目中快速集成 EasyModal。

基础用法

// 简单提示 await EasyModal.alert({ title: '提示', content: '操作成功！' }); // 确认对话框 const result = await EasyModal.confirm({ title: '确认', content: '确定要删除这个文件吗？' }); if (result.confirmed) { console.log('用户确认了操作'); }

高级用法

// 高级配置 await EasyModal.confirm({ title: '高级配置', content: '这是一个配置了多个选项的弹窗', theme: 'dark', width: 400, countdown: 10, confirmText: '同意', cancelText: '拒绝', onConfirm: async () => { console.log('用户点击了同意'); } });

在插件中使用

在浏览器插件中，你可以这样使用：

// content script 中 import { EasyModal } from 'easymodal'; // 或者直接引入UMD版本 // <script src="easymodal/dist/index.js"></script> // const EasyModal = window.EasyModal; // 显示弹窗 EasyModal.alert({ title: '插件通知', content: '页面加载完成！' });
